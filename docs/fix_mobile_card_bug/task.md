# タスク：モバイルでのカード消失バグおよびクリック領域の改善

## 現在の状況
- モバイルで "?" マーク（ヘルプ）を押すとカードが消失する。これは `Dashboard.tsx` が IntersectionObserver を使って付与した不可視状態解除クラス (`card-visible`) が、React の再レンダリングによって上書きされ失われるためである。
- "?" マークのクリック領域が小さすぎて操作しづらい。

## ゴール
- カード内のインタラクション（ヘルプ表示など）を行ってもカードが表示されたままにする。
- "?" マークのクリック領域を拡大し、ユーザビリティを向上させる。

## アプローチ
1. **カード消失の修正**:
   - `Card.tsx` 内で `className` プロパティがステート変更 (`showInfo`) に依存しないように変更する。
   - `z-index` の制御（`z-40` クラスの付与）を React の宣言的な `className` から、`useRef` と `useEffect` を用いた命令的な操作に変更する。
   - これにより、再レンダリング時に副作用として外部から付与されたクラス（`card-visible`）が React によって削除されるのを防ぐ。

2. **クリック領域の拡大**:
   - ヘルプボタンにパディング (`p-2`) を追加し、ネガティブマージン (`-m-2`) でレイアウトへの影響を相殺する。
